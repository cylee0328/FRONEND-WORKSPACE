텍스트 관련 스타일(text~)

color : 텍스트의 색상을 지정

        선택자 {
            color : 색상명/16진수/rbg. rgba, hsl..
        }

text-decoration : 텍스트에 줄을 긋거나 줄을 없앨때 사용

        선택자 {
            text-decoration : none/underline/overline/line-through;

        }
  • underline : 텍스트 영역 아래에 줄 긋기
  • line-through : 텍스트 영역 중간에 줄 긋기
  • overline : 텍스트 영역 위에 줄긋기
네이버로 가자

text-transform : 영문 텍스트의 대소문자 변환시 사용

        선택자 {
            text-trandform:uppercase/lowercase/capitalize;
        }
  • uppercase : 모든 영문자를 대문자로
  • lowercase : 모든 영문자를 소문자로
  • capitalize : 영문자의 첫글자만 대문자로

text-align : 텍스트를 정렬할 때 사용

        선택자 {
            text-align : left(기본값)/justify/right/center;
        }

양쪽정렬

예수는 이상의 미묘한 꽃이 인생을 약동하다. 더운지라 같이, 전인 가는 열락의 것이다.보라, 있음으로써 운다. 많이 이 무한한 보이는 무엇을 시들어 천지는 피다. 꾸며 타오르고 공자는 우리의 품었기 바이며, 아니더면, 피다. 우리 주는 밥을 새가 날카로우나 이상을 천지는 봄바람이다. 원질이 청춘을 못할 피에 대한 놀이 수 그들은 부패뿐이다. 소금이라 대고, 노년에게서 오아이스도 꽃이 있는 그들에게 피부가 전인 것이다. 생생하며, 없으면 어디 구할 청춘의 풀밭에 목숨이 사막이다. 열락의 공자는 찾아다녀도, 얼음과 든 끝까지 귀는 되는 돋고, 칼이다. 무엇을 사랑의 보는 원질이 인생에 봄바람이다. 얼음과 청춘 못하다 공자는 같이, 반짝이는 봄바람이다. 그러므로 가진 보이는 천지는 몸이 가치를 주는 없으면, 이는 있으랴? 시들어 가치를 역사를 보라. 그들의 끓는 방지하는 든 것이 굳세게 아니다. 속에 같이, 것은 있는 피부가 유소년에게서 투명하되 예가 곧 말이다. 그들은 이상은 인도하겠다는 이것이다. 끝에 있는 실로 커다란 천하를 피가 가진 못할 오아이스도 그리하였는가? 굳세게 이상을 장식하는 철환하였는가? 할지니, 곳으로 눈이 커다란 그리하였는가? 봄날의 그들에게 하는 소금이라 없으면, 이상은 새 있는 꽃 있다. 대한 청춘 것은 커다란 보이는 이상이 쓸쓸하랴? 아니더면, 별과 능히 몸이 인간은 꾸며 피는 열매를 이 것이다. 풍부하게 가진 그림자는 과실이 되려니와, 철환하였는가? 찬미를 가슴에 끓는 맺어, 긴지라 것은 같이, 거선의 우는 봄바람이다. 설산에서 굳세게 행복스럽고 찾아 것이다. 같은 시들어 풀이 낙원을 듣기만 거선의 가치를 피어나는 부패뿐이다. 우는 무엇이 천고에 청춘의 쓸쓸한 주며, 때문이다. 긴지라 같은 하여도 현저하게 보이는 없는 구하지 피가 것이다. 넣는 사라지지 충분히 뜨거운지라, 갑 천하를 같은 것이다. 그들의 피는 만물은 보는 못할 것은 그들은 속에 옷을 있는가?

오른쪽정렬

귀는 내는 것은 장식하는 인류의 행복스럽고 것이다. 대고, 위하여, 위하여 것은 청춘의 천고에 위하여서. 뭇 인생에 꽃이 얼마나 구하기 뼈 피고 귀는 것이다. 반짝이는 뜨거운지라, 것은 충분히 할지니, 우리의 위하여, 보이는 품에 보라. 청춘의 간에 갑 이것이야말로 소금이라 피가 용기가 하였으며, 그리하였는가? 두기 돋고, 커다란 맺어, 이것이다. 사람은 찾아 거친 청춘의 그리하였는가? 싹이 꽃이 설산에서 앞이 품으며, 싸인 가슴에 그것을 보라. 붙잡아 같은 불어 인도하겠다는 있을 현저하게 가지에 것이다. 굳세게 피어나기 피가 전인 있으며, 듣기만 보이는 희망의 있는가? 같이, 관현악이며, 피부가 천지는 있음으로써 봄바람이다. 석가는 미묘한 대중을 과실이 생의 사람은 하여도 끓는 이것이다. 같이 이상 인류의 보이는 힘차게 바로 품고 부패뿐이다. 황금시대의 귀는 청춘의 설레는 수 뜨고, 있다. 무한한 위하여서 곳이 봄바람을 그들을 있는가? 고동을 인간은 원질이 있다. 같으며, 부패를 그들의 품에 용기가 모래뿐일 약동하다. 이상 바로 주며, 만천하의 힘차게 사는가 따뜻한 같지 능히 힘있다. 우리는 이상은 어디 주는 꽃이 용감하고 있는 피고 아름다우냐?

가운데정렬

바이며, 뼈 이상, 이것이야말로 수 청춘을 주는 거친 어디 교향악이다. 예가 천지는 그들은 석가는 만물은 인생을 청춘 희망의 말이다. 봄날의 무엇을 가슴에 위하여서 청춘의 내는 그들의 있는가? 살 밥을 품었기 구하지 피에 설산에서 꾸며 커다란 하여도 그리하였는가? 봄바람을 듣기만 같은 위하여, 못할 봄날의 피다. 기쁘며, 눈에 고행을 역사를 설산에서 얼음 이것이다. 같으며, 그들을 옷을 것이다. 관현악이며, 평화스러운 못할 인생을 붙잡아 못할 것이다. 청춘의 커다란 피는 고동을 얼음과 피고 이상 있다. 곧 있는 피가 시들어 사랑의 아름다우냐? 가치를 가슴이 바이며, 할지니, 인생을 인생의 청춘의 광야에서 황금시대다. 유소년에게서 대고, 인생을 봄바람이다. 예가 우리 만천하의 속에 이 착목한는 아니더면, 목숨을 말이다. 없는 기쁘며, 대중을 봄날의 길지 것이다. 피고, 동력은 우리 얼음과 뜨거운지라, 아니다. 할지라도 하는 것이다.보라, 뭇 피어나기 소담스러운 황금시대다. 두손을 속에 구하기 사는가 되는 대고, 것은 부패뿐이다. 거선의 하였으며, 그들의 시들어 가는 작고 없으면 것이다. 크고 이상 희망의 뼈 가는 꽃이 공자는 있는가? 청춘의 피고 피에 풀이 따뜻한 우리 봄바람이다.

text-shadow : 텍스트에 그림자효과를 줄 때 사용

        선택자 {
            text-shadow: 가로거리(x) 세로거리(y) 번짐정도 색상;
        }

        - 번짐정도와 색상을 생략 가능
        - 가로거리와 세로거리가 양수일 경우 오른쪽 아래에 그림자 생성
        - 가로거리와 세로거리가 음수일 경우 왼쪽 위에 그림자 생성
        - 번짐정도가 0일 경우 글자 색이 진하다. 
        - 색상은 기본값이 글자색과 같다.
